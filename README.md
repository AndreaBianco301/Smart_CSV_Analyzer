# 📊 Smart CSV Analyzer

A simple interactive data analysis tool built with Streamlit.

## 🚀 Features
- Dynamic pivot tables
- Interactive Plotly charts
- Multi-dimensional grouping
- Data filtering system

## 🛠️ Technologies
- Streamlit
- Pandas
- Plotly
- Python

## ▶️ Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
